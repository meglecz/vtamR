RandomSampleFastaR
	cross platform
	count sequences by reading files by chunks
	random sample by reading files lines by line
	file compression is determined gzfile or file for read/write files
RandomSampleFastaLinux (equivalent of RandomSampleFastaR)
	only linux
	count sequences by bash command
	random sample by rvsearch
	if compress output => smart_gzip

RandomSample
A wrapper to deal with files in a loop and determine I/O filenames 
then run RandomSampleFastaR or RandomSampleFastaLinux foreach file
	Input/output can be compressed or uncompressed
	Files are read and written without explicit compression/decompression
